# Disclaimer

WindowsDnsSwitcher is provided for managing DNS settings on the user's own Windows device.

Users should only use it on devices and networks they are authorized to manage. Changing DNS settings may affect network connectivity. Restore Automatic DNS Mode if a network stops working after configuration changes.

The author is not responsible for misuse, incorrect configuration, or network access issues caused by changing DNS settings.

Diagnostic website shortcuts are provided only for convenience. They open third-party websites in the default browser, and those websites may receive public IP address, browser, and network information.
