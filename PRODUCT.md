# Product

## Register

product

## Users

Windows users who need to switch a physical WLAN/Wi-Fi adapter between automatic DNS and a saved custom IPv4 DNS profile.

## Product Purpose

WindowsDnsSwitcher provides a small, local desktop utility for changing only the selected adapter's IPv4 DNS mode with minimal risk and clear feedback.

## Brand Personality

Calm, utilitarian, trustworthy.

## Anti-references

Avoid marketing-style layouts, decorative UI, cramped dense controls, oversized display text, and ambiguous buttons that hide what will happen.

## Design Principles

- Prioritize the two DNS mode actions.
- Keep adapter selection legible before any network mutation.
- Use compact but not cramped spacing.
- Make warnings clear when the selected adapter may be virtual.
- Keep visual styling quiet so the tool feels like a reliable Windows utility.

## Accessibility & Inclusion

Use readable Chinese labels, sufficient contrast, keyboard-reachable standard controls, and stable layout dimensions that do not overlap under Windows DPI scaling.
